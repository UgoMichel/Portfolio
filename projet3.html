Context

Sujet

Une application web qui permet de jouer en ligne au jeu de société “Les Aventuriers du Rail” version Europe.

Les Missions

Le projet avait pour mission de réaliser une application web qui permet de jouer en ligne au jeu de société “Les Aventuriers du Rail” version Europe. Le but du projet était de développer des compétences avec le langage Java. Il fallait utiliser toutes les notions de base de ce langage pour programmer le côté backend du projet. Le code devait respecter dans les moindres détails les règles officielles du jeu. Par ailleurs, en ce qui concerne l’interface du projet, il fallait maîtriser le langage JavaFx. Ainsi, l’objectif était d’utiliser tout ce que JavaFx peut mettre à disposition comme les listener, etc.

Mesure & Optimisation

Cette application devait suivre certains critères comme l’utilisation de Java orientée objet et une ergonomie qui est construite autour d’un parcours utilisateur bien précis.

Composants à intégrer

Les notions qui nous étaient demandées, étaient de concevoir un algorithme en Java qui devait suivre dans les détails les règles du jeu de société. Puis d’intégrer la carte interactive du jeu au centre de l’application. Enfin, chaque groupe devait trouver une façon originale d'implémenter tous les éléments de jeu (pioche, cartes…) autour de cette carte.

Langages autorisés

Java, JavaFX

Analyse

Lors de ce projet, j’ai pu développer et apprendre de nouvelles compétences techniques en informatique.

Côté Java

À travers ce projet, j’ai pu utiliser toutes les notions de base de Java orientée objet. C’est notamment avec ce projet que j’ai implanté un grand nombre de classes. Par conséquent, j’ai dû utiliser des classes abstracts, mais aussi des interfaces. De plus, la notion d’héritage était au centre du projet. Par conséquent, ce projet m’a permis de comprendre l’objectif de principes comme le polymorphisme.

Côté JavaFX

En ce qui concerne le JavaFX, c’est grâce à ce projet que j’ai découvert ce langage. J’ai appris et appliqué les bases de ce langage. JavaFX possède une hiérarchie bien spécifique, qui commence de la Scène en passant par des Pane qui eux possèdent des Button. De plus, ce langage m’a permis de concevoir des listeners pour que l’application s'adapte visuellement en fonction de certains événements. On peut prendre l’exemple de pop-up qui indique des informations à l’utilisateur. Enfin, JavaFX m'a permis de réaliser une interface qui respect le parcours utilisateur et une ergonomie bien réfléchie.

Amelioration

Pistes d’amélioration

Pour les pistes d'amélioration, on peut penser à la création d’un système de compte utilisateur. Ce système permettrait à chaque utilisateur de posséder un espace de profil. Par conséquent, l’utilisateur pourrait avoir un historique de ses parties. Puis on peut aussi penser que l’utilisateur pourrait perdre ou gagner des points en fonction de son positionnement à la fin de chaque partie. Tout cela serait réalisable en utilisant du PHP et en reliant l’application avec une base de données.

Ce qui serait vraiment intéressant est de créer un launcher dans lequel nous pourrions choisir de jouer à la version Europe ou à la version Monde du jeu "Les Aventuriers du Rail”.

Comme toujours, une version mobile sera toujours la bienvenue.
